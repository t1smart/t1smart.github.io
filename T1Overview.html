Thomson ONE Smart Developer Integration

Thomson ONE Smart makes it easy to integrate your firm’s proprietary and third-party web applications into our platform. Your web app can exchange symbols, lists, user data, and other information and instructions with our framework and the other services that it contains.

If your web application can run in Internet Explorer, it can run within Thomson ONE with no coding changes.

Benefits and Capabilities:

- Your firm’s proprietary application or web app can pass/update ticker symbol, account number and other component interactions to every application on screen.

- Automatic translation of one symbol type to another ( e.g. Ticker to CUSIP )

- Add custom left and right-click menu options

- Custom Quote (Ratings and Restrictions info) implementation to our Quote/Monitor application via SFTP/FTP.

- Display real-time market data within your web application.

- Create a drilldown (hyperlink) in your application to open other Thomson ONE apps while optionally sending relevant context to it. Every application available to the user through our framework is a potential drilldown target.

Features:

Adding your App to the Right Click Menu

Ability to inject items into our right click mouse menu

Symbol passing

Ability to add your application inside our framework and accept context passing.

Adding a link to your application

Add a drilldown link in your application which opens a Thomson ONE Smart application
